## 📌 Convention

### 1️⃣ 소스코드 파일 이름 규칙
각 코드 별 목적을 주석으로 작성합니다.
변수와 함수 이름 또한 역할을 알 수 있도록 간단한 주석을 덧붙입니다.
- 문제출처_문제번호.java
- `BOJ_1234.java`
- 문제출처 작성 통일
    - BOJ - 백준
    - SEA - 삼성 SW Expert Acadamy
    - PGS - 프로그래머스
